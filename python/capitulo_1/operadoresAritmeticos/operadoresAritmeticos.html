concatenacion

bueno ya sabiamos sobre los operadores aritmeticos pero dentro de todo esto tambien llega la concatenacion que es muy importante para poder unir cadenas , tenemos 2 formas de concatenar bueno en realidad hay mas pero te mostrare los mas faciles/usados para esta tarea.

concatenacion tradicional

para usar la concatenacion tradicional se usa "+" ahora un ejemplo del como se usa:
"cadena " + "otra cadena";
este se puede aplicar para "" y """ """.
