ruta para ser desarrollador

aqui te dare la ruta para que puedas ser un desarrollador web y asi puedas conseguir un trabajo como desarrollador.

html

esta herramienta nos sirve para estructurar los sitios web para poder entender mejor podemos decir que html es el esqueleto de una pagina web por ejemplo si a este sitio web borras el estilo y la funcionalidad veras como esta estucturado este sitio probablemente veas muchas letras todas feas y sin colores sin funcionalidad un elemento ensima de otro pero esta bien que se vea asi ya que solo es la estructura.
html no es para estilizar y tampoco intentes estilizar un sitio web con html ya que eso esta mas por eso es que en una parte parte de los apuntes dije que la etiqueta center no se debe utilizar ya que html no es para estilizar.

css

con esta herramienta vamos a estilizar los sisitos web en simples palabras es como ponerle ropa o accesorios para que el sisio web se vea bien esta herramienta trabaja en conjunto con javaScript para poder mover elementos para cambiar dinamicamente muchas cosas pero esto no se lograria sin css , con css se pueden hacer cosas muy geniales con las transisiones , animaciones , etc. css tiene una de las mejores propiedades que es el responsive design que es para que nuestro sitio web se adapte a distintas resoluciones de pantallas , amobiles , pc , laptobs , tv , etc.

javaScript

esta herramienta es para poder dar dinamismo a un sitio web eb simples palabras es como si fuesen los organos de un sitio web esta herramienta hace que sea posible crear una funcionalidad al sitio web , esto trabaja en conjunto a html y css ya que este lenguaje fue hecho para el desarrollo web pero este lenguaje se puede sacar de la web gracuas a node.js pero ese es tema aparte muy recomendado de aprender este lenguaje.
