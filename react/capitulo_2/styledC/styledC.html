styled component

esta es una libreria demasiado util para trabajar con los estilos ya que hacemos todo desde 1 archivo sin crear nada y podemos darle logica a los estilos.

instalacion

para poder instalar esta libreria debemos acceder a nuestro proyecto de react osea en la misma ubicacion donde ponemos "npx start" para iniciar el server bien , alli debemos poner lo siguiente.
esto instalara la libreria.

npm install --save styled-components

una vez se instalo la libreria podemos importar nuestra libreria donde queramos

import styled from "styled-components";

ThemeProvider

ThemeProvider es un componente de styled-components que permite asignar un tema unificado a todos los componentes hijos en una aplicación, lo que resulta en una fácil gestión y reutilización de estilos, y la posibilidad de cambiar el tema en tiempo de ejecución.
para usar ThemeProvider debemos de importar esta funcionalidad.

import styled {css,keyframes,ThemeProvider} from "styled-components";

esto es como un contenedor donde podemos poner un objeto con el estilo que queremos que se pongan.
