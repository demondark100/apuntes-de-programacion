hooks personalizados

esto nos sirve para poder crear una funcion que usaremos muchas veces en nuestra aplicacion de react.

pasos par usar los hooks personalizados

  1. crear una nueva carpeta
  2. al archivo que creemos siempre su nombre debe iniciar con use ejemplo "useFunction" o "useComponent" puede ser cualquier nombre solo asegurate que lleve use para que react detecte que es un hooks personalizado.

ejercicio

hacer una funcion que haga una peticion con fetch , recuerda que ya hicimos esa funcion podemos reusarla pero quitaremos la logica de ingresar las imagenes , el nombre y la funcion solo llamara a la API y controlaremos los posibles errores.
